About osu!droid

osu!droid is an unofficial osu project for android. This game was created many years ago by the osu! community. It is a free-to-play rhythm game for Android devices that involves clicking circles. Currently, a small group of fans and several foreign friends are working on this project together. Similar to it's original PC version, osu!droid's gameplay involves players tapping, sliding, and spinning on circles, sliders, and spinners to the beat of music. Mobile gamers that like difficult rhythm games are fond of this game. The osu! community has totally created a large number of songs and maps, which can be imported to osu!droid, and this game only has one game mode, including the osu!standard game mode.

Currently, osu!droid will be having it's last update as of June, which means the developers are currently working on it's new, overhauled version, with the name of rimu!. It's source code was entirely made in Java, a popular programming language which is capable of making mobile, and computer softwares. The goal of the game is to develop new features that we're only exclusive in-game. The lead Developer, Rian8337 (Rian) has worked hard making lots of game mechanics to make it work like the original osu! on PC. Some mechanics in-game were pretty much different, from the performance points (pp), in-game exclusive mods, different server, among many others.

osu!droid will be going to discontinue soon, but there are many players currently playing in. Since the game will remain untouched sooner, rimu! will be focused for development as it was being made by one the developers from the same game, also known with the name of reco1l.
