# osudroid!bench
<img src="/medias/osulogo.svg">

## All about this project

osudroid!bench is an **osu!droid** related web project, which contains all phones with the results you can compare with. This project is made by only one contributor.

It's webpage contains a table, containing phones you can compare with, *actually there are a few, please submit your benchmark score please*

## Note
This project is related to osu!droid, which can be discussed on the [project's discord server](https://discord.gg/kUaSggQP7m)

## Scripts
Please take note that this project mainly uses javascript, which can be executed mainly at the start of the project, which means that most features are contained on this HTML project.

## Submitting Scores
Wanted to see your phone model in the website? Please submit your very own score by [clicking here](https://forms.gle/1vbqbEbiLirCobmC6) and complete the survey in order to submit your benchmark score.

*please note that phones with low benchmark scores will be replaced with higher ones*
